## ADORe Simulation Scenarios
A ROS2-based repository containing standardized simulation scenarios for
testing and validating autonomous driving algorithms within the ADORe
(Automated Driving Open Research) platform. This collection includes highway
scenarios (lane keeping, overtaking, merging), urban environments (traffic
lights, pedestrian interactions), and intersection challenges (unprotected
turns, roundabouts), each designed to evaluate specific aspects of autonomous
vehicle behavior with automated metrics tracking for trajectory accuracy,
safety, and comfort.

These scenarios are designed to be run in the ADORe framework.
